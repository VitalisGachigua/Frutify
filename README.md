# Frutify
This simple fruit website to demo Tailwind Css capabilities

## Pages
1. Home Page
    -Hero Section
    -Featured Fruit cards
    -Testimonials
    -call to action buttons(CTA)

2. Shop/Catalogue(products page)
    -Multi-column layout
    -Sidebar Filters
    -sale badges - add to cart
    
3. Product Detail 
    -Large galleries
    -quantity selectors
    -related items

4. Cart/checkout
    -Order summary tables
    -input fields
    -sticky checkout buttons

5. Responsive navbar and footer
